Further Details

This is a project that incorporates an artificial intelligence agent using reinforcement learning (Q-Learning) that plays the snake game and gets better at it iteratively.

Q-Learning algorithm

The code below shows how I implemented the Q-Learning algorithm in my project as I based my implementation on the bellman equation.

 void UpdateQTable(int currState,int action, double reward,int nextState)
                {
               
                   double currentQ = qtabledict[currState][action]; 
                  
                   double maxQ = qtabledict[nextState].Values.ToArray().Max();
                   
                   double newQ = currentQ+ learningRate * (reward + (discountFactor * maxQ)-currentQ);
               
                   qtabledict[currState][action] = newQ;
               
                }

States

States are a vital part of the q learning algorithm as it acts as a guide for the agent while navigating the environment. I utilized an enum to declare the different states and a big advantage of using enums is the fact that you can assign different values to each variable.

Actions

In order for the agent to be able to navigate through the environment it needs the ability to take actions and as discussed before, the actions are basically the controls. The choose action function decides what the best action is through the exploration vs exploitation method.

  int ChooseAction(int currentState)
                    {
                       
                        float random =UnityEngine.Random.Range(0.0f, 1f);
                        int action;
                        int bestAction = 0;
                        if (random < explorationRate)
                        {
                            // 0-3
                            bestAction = UnityEngine.Random.Range(0, 4);
                        }
                        else
                        {
                            // 0-3
                            double bestQVal = -1e9;
                            for (action = 0; action < 4; action++)
                            {
                              
                                if (qtabledict[currentState][action] > bestQVal)
                                {
                                    bestQVal = qtabledict[currentState][action];
                                    bestAction = action;
                                }
                                
                               
                            }
                        }
                   
                        return bestAction;
                    }
