JavaScript


Hamburger

Visibility

This was my first interaction with JavaScript as I used it to create a hamburger menu that contained the main parts of the site such as Home, My Services, About Me, and My Work. To make the hamburger work the ".nav-toggle" class name will be assigned to a button in html and a reference of it will be stored in the constant variable navToggle. Then an event listener will be added to navToggle. if the assigned button is clicked the arrow function will be executed which sets the visibility of the hamburger menu.


Links

navLinks does the exact same thing as navToggle except that is selects all the elements that have been assigned ".nav__link". Then, a forEach loop that states for each link in navLinks an event listener gets added to the link and when the link is clicked the arrow function is executed.The function removes "nav-open" from the body element which closes the hamburger.
